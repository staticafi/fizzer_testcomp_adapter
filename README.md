Contents
========

This README.md file describes the content of the tool, i.e., what is in this 
directory and sub-directories.

The license under which this tool is released is in the file ./LICENSE.txt.

The actual tool is under directory `./fizzer/`. The files in the root
directory represent only an adapter for plugging the tool into the
Test-Comp's pipeline. 

Usage of the tool (via the adapter) is show-cased by the `./smoketest.sh`.

The source code of the tool is available here: https://github.com/staticafi/fizzer
The source code of the adapter is available here: https://github.com/staticafi/fizzer_testcomp_adapter
